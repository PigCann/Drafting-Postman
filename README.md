## 🎮 怎么玩

双击 `《漂流信使》Drifting Postman.html` → 浏览器打开 → 阅读开场故事 → 启航

| 按键 | 作用 |
|------|------|
| `W A S D` / 方向键 | 航行 |
| `空格` / 点击 | 捞信、靠岸、推进对话 |
| 右上角 `🌐` | 中英切换 |

每封信读完后根据意象线索判断方向（北/南/东/西）。选错可以一直重试，答错会弹出 💡 提示。12 封信全送达 + 3 座岛对话 → 通关。

---

## ✍️ 贡献你的信

每一封信就是一个 JS 对象。打开 HTML 文件，找到 `<script>` 标签内的 `ltr` 数组，按格式新增：

```javascript
{f:'发件人身份 · 地点', t:'"信的内容。"', d:'方向', h:'答错时的引导提示'},
```

- **f** — 发件人身份 + 地点，如 `'渔夫 · 智利 · 瓦尔帕莱索'`
- **t** — 信的内容，120 字以内为佳，方向线索藏在文字里
- **d** — 正确方向，`'n'` / `'s'` / `'e'` / `'w'`
- **h** — 答错后的引导提问，帮助玩家推理

### 方向线索设计原则

不用直接说「她在北边」，用常识暗示：

| 方向 | 暗示方式 |
|------|---------|
| ⬆️ 北 | 更冷、更早入冬、极光 |
| ⬇️ 南 | 更暖、有夏天、南十字星 |
| ➡️ 东 | 更早看到日出、太阳先升起 |
| ⬅️ 西 | 太阳落下、海风从日落来 |

### 提交方式

1. Fork 本仓库
2. 在 `ltr` 数组中添加你的信
3. 提交 Pull Request

中英文版本的 `ltr` 都要添加（中文在 `L.zh`，英文在 `L.en`）。

## 🔮 未来计划

### 信件池 · 每次都是新的海

目前游戏内置 12 封信。未来的版本中，信件数量不再设上限——所有贡献者提交的信将汇入一个**公共信件池**。每位玩家打开游戏时，系统从池中随机抽取 12 封。你可能遇到渔夫、面包师，也可能遇到一位从未谋面的陌生人写下的句子。每一次启航，海上的信都不完全相同。

### 随机身份 · 你在海上的名字

现在的主角叫林渡，英语专业大四学生。未来版本将支持**随机生成玩家身份**——你可能是一个刚被 AI 取代了工作的设计师，一个在深夜值班的护士，一个退休后在老年大学学写字的老人，一个每天送外卖、在等单间隙读诗的骑手。

你不是被固定的角色。你是任何一个——曾经在词语间犹豫过、被打动过、觉得"有些东西说不清楚但还是想试试"的人。

### 治愈的不只是英语系

AI 冲击的不只是翻译行业。设计师看到 Midjourney，程序员看到 Copilot，配音演员听到 AI 生成的自己的声音。但总有一些东西是通用的——一句话里的体温，一封信里的迟疑，一段不完美却真诚的表达。

这个游戏的底色不是"英语专业有多难"，而是——**在这个越来越快的时代，还有人愿意为一个词犹豫，为一句看不懂的话停下来，为千里之外一个陌生人写下的不完美句子——心头一热。**
